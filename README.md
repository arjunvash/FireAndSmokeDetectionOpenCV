# 🔥 Fire and Smoke Detection System

An AI-powered computer vision project that detects **fire and smoke in real-time** using deep learning techniques. This system helps in early detection of fire hazards to improve safety and reduce damage.

---

## 🚀 Features

- 🔍 Detects **Fire**, **Smoke**, and **No Fire/Smoke**
- 📷 Works with images, videos, and live webcam
- ⚡ Real-time detection using OpenCV
- 🧠 Deep Learning model (CNN)
- 📊 High accuracy classification

---

## 🛠️ Technologies Used

- Python 🐍
- OpenCV
- TensorFlow / Keras
- NumPy
- Matplotlib
- Google Colab

---

## 🧠 Model Details

- Model Type: Convolutional Neural Network (CNN)
- Classes:
  - Fire 🔥
  - Smoke 💨
  - No Fire 🚫
- Training done using labeled dataset
